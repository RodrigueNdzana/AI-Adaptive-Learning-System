## Explanation
### Key Actors & Roles
- Student – Uses system to learn and track progress
- Student with Disability – Uses accessibility features
- Educator – Monitors progress & provides content/feedback
Disability Support Officer (DSO) – Ensures accessibility compliance
- Admin – Manages system and generates reports
- AI/ML Engineer – Maintains AI model
### Relationships

AI recommendations depend on AI engine
Content upload includes accessibility validation
Progress monitoring includes risk detection
 Stakeholder Alignment
Supports students → personalized learning
Supports educators → identify struggling students
Supports DSO → accessibility enforcement
Supports admins → reporting & compliance
