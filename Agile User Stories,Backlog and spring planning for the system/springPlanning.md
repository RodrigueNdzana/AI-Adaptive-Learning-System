# Sprint Planning (2 Weeks)

##  Sprint Goal

Deliver a functional MVP that allows students to log in, access accessible content, view progress, and receive AI-based recommendations.

---

##  Selected Sprint User Stories

* US-001: Login/Register
* US-002: Learning Profile
* US-003: AI Recommendations
* US-004: Content Access
* US-005: Dashboard

##  How Sprint Supports MVP

This sprint delivers:

* **Core user access** → Login system (US-001)
* **Personalisation** → Profile + AI recommendations (US-002, US-003)
* **Learning delivery** → Multi-format content access (US-004)
* **Tracking & insights** → Progress dashboard (US-005)

---

##  Summary

By the end of this sprint, the system will provide a **functional MVP** where users can:

* Log in securely
* Receive personalised learning experiences
* Access content in multiple formats
* Track their learning progress
