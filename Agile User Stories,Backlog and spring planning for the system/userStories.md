# User Stories

| Story ID | User Story                                                                                         | Acceptance Criteria                         | Priority |
| -------- | -------------------------------------------------------------------------------------------------- | ------------------------------------------- | -------- |
| US-001   | As a student, I want to register and log in so that I can access the system securely.              | Login ≤ 3 sec, lock after 5 failed attempts | High     |
| US-002   | As a student with disability, I want to create a learning profile so that content is personalised. | Profile saved, preferences applied globally | High     |
| US-003   | As a student, I want AI recommendations so that I get relevant learning materials.                 | ≥3 recommendations, match score shown       | High     |
| US-004   | As a student, I want to access content in different formats so that I can learn effectively.       | Video, audio, text available                | High     |
| US-005   | As a student, I want to view my progress dashboard so that I can track performance.                | Dashboard loads ≤2 sec                      | High     |
| US-006   | As an educator, I want to monitor student progress so that I can support struggling students.      | Filter + at-risk highlighting               | High     |
| US-007   | As an educator, I want to upload content so that students can access learning materials.           | File validation + upload success            | Medium   |
| US-008   | As a system, I want to detect at-risk students so that early intervention is possible.             | Alert triggered if performance drops        | High     |
| US-009   | As a student, I want to receive feedback so that I can improve my work.                            | Notification within 1 min                   | Medium   |
| US-010   | As a student, I want accessibility settings so that I can customise my experience.                 | Settings applied globally                   | High     |
| US-011   | As an admin, I want reports so that I can monitor system usage.                                    | Reports generated ≤10 sec                   | Medium   |
| US-012   | As a system admin, I want data encrypted so that security compliance is met.                       | AES-256 encryption enforced                 | High     |
